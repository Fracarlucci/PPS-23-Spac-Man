# Spac-Man

## Introduzione

Il progetto prende spunto dal celebre gioco [Pac-Man](https://it.wikipedia.org/wiki/Pac-Man), con l'obbiettivo di riprodurne una versione simile.

Di seguito sono riassunti i principali elementi e funzionalità dell'applicazione:

- Mappa con muri, teletrasporto e puntini da mangiare
- Giocatore con possibilità di muoversi all'interno della mappa
- Quattro fantasmi (nemici) con movimento autonomo
- Punteggio giocatore
- Gestione game win/over

Funzionalità addizionali, nel caso avanzi del tempo:

- [ ] Oggetti che permettono di mangiare i nemici (cambiamento di stato dei fantasmi)
- [ ] Vite del giocatore ed oggetti per recuperarle
- [ ] Comportamenti differenti per ciascun fantasma
- [ ] Altre mappe
